# Getting The Level
This part of the guide will instruct you on how to obtain the **sticker** to connect to the server.

## 1 - Preparing the files
1. Join Zhaxxy's discord server. Linked [here.](https://discord.gg/YGfHwAB928)
2. Agree to the rules and go to the eZlbp2ps4-bot-usage channel.
3. Type `/lbponline_public_beta` into the text box.
4. Enter your PSN username into the Account ID box.
5. Click on Game ID and select the region of your LBP game.
6. Press send and wait for the bot to finish.
7. When the bot finishes, download the .zip file the bot sent.

## 2 - Setting up the USB stick
1. Plug the USB stick into your device.
2. Make sure your USB stick is formatted as FAT32 or exFAT.
3. Extract the .zip file you downloaded earlier, and copy the `PS4` folder inside onto the USB stick.

You now have the level on your USB drive, now it's time to get the sticker!
